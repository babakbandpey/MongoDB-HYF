# MongoDB-Lesson-1
ManogoDB Lesson 1 for Hack Your Future ***Class12***

## MongoDB not RDBMS

## Not a Transactional DB

## Data is saved in JSON format but in Binary not Text, BSON (Bee-SON)

```
{
    "firstname": "Peter",
    "lastname": "Membrey",
    "numbers": [
        {
            "phone": "+852 1234 5678"
        },
        {
            "fax": "+44 1234 565 555"
        }
    ]
}
``` 

## There are no tables vs. RDBMS which has a static design
